**Apakah yang dimaksud dengan Variabel dan Tipe Data didalam Javascript? Jelaskan jenis-jenis nya!**

---
```
VARIABEL adalah tempat untuk menyimpan data atau nilai yang dapat di ubah selama program berjalan.

TIPE DATA adalah jenis nilai yang dapat disimpan dalam VARIABEL
CONTOH BEBERAPA TIPE DATA :
Number: digunakan untuk menyimpan nilai numerik seperti angka bulat dan desimal.
String: digunakan untuk menyimpan teks atau kumpulan karakter.
Boolean: digunakan untuk menyimpan nilai kebenaran, yaitu true atau false.
Null: digunakan untuk menunjukkan bahwa variabel tidak memiliki nilai atau objek yang tidak terdefinisi.
Undefined: digunakan untuk menunjukkan bahwa variabel tidak memiliki nilai yang ditetapkan atau tidak didefinisikan.
Object: digunakan untuk menyimpan struktur data kompleks yang terdiri dari properti dan metode.
Array: digunakan untuk menyimpan sejumlah nilai dalam satu variabel dan diakses dengan indeks numerik
